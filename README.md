# MMD Jiggle Bones

RGBA式胸部物理移植插件，可将RGBA式胸部物理移植到任意含胸部物理的模型中。

## 使用前提
- 模型本身含有胸部骨骼及胸部物理。
- 模型本身含有“上半身2”骨骼。
- （可选）腕、ひじ、手首骨骼拥有对应的追踪骨骼类型的刚体。

## 使用限制
- 生成的模型无法在Blender、MMM、NexGiMa中直接进行烘焙，如需烘焙，请使用MMD桥生成带物理的VMD文件，或采用ABC流程。

## 执行流程
1. 填写“源模型地址”、“权重比例”、“刚体比例”后，点击执行。
2. 使用插件生成的模型进行后续流程。

## 注意事项
- 不建议将已生成的模型再次作为输入进行处理，如需调整，请始终使用原始模型作为输入。
- 若胸部与胸饰非物理部分穿模，请调小权重比例，或自行修改/隐藏胸饰。
- 若提示未找到胸部骨骼，但模型确实存在胸部物理，可在PE中将胸部骨骼重命名后再导入尝试（命名格式：左胸1、左胸2、...、右胸1、右胸2、...）。
- 模型太小或太大均会产生问题，请在正常角色身高（0.9m~2.5m）下进行操作。
  - 若调整权重比例后胸部物理依然下坠明显，可尝试先放大模型后再执行插件。
  - 若调整权重比例后胸部物理依然晃动不明显，可尝试先缩小模型后再执行插件。
  - 不建议将已生成的模型进行缩放，如需缩放，请在执行插件前处理。
- 胸部刚体群组默认为15，若其它物理刚体的群组也为15，则可能会造成穿模。（由于编号起始顺序不同，在Blender中默认为14）
- 若调整刚体比例后依然存在碰撞或非碰撞，请自行在PE中调节碰撞组。
